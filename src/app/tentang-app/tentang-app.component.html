Aplikasi web yang berisi daftar kontak layanan kesehatan berupa alamat, nomor telepon, dan lokasi peta GPS untuk layanan Pusat Kesehatan Masyarakat (Puskesmas), Rumah Sakit Umum, dan Rumah Sakit Khusus, yang berada di kota DKI Jakarta dan sekitarnya. Semua kategori tersebut dapat diakses dengan menekan menu-menu yang tersedia di bagian atas halaman ini.

Data kontak layanan kesehatan dan medis ini bersumberkan dari data yang dihimpun oleh Jakarta Smart City, yang dapat dipakai melalui bantuan Application Programming Interface (API) yang tersedia di http://api.jakarta.go.id/

API Jakarta merupakan sebuah layanan yang menyediakan informasi data yang ada di lingkup Pemerintah Provinsi DKI Jakarta. Data ini mencakup semua bidang yang ada di Provinsi DKI Jakarta seperti Kesehatan, Kebersihan, Transportasi, dll. Dengan adanya API Jakarta, maka para developer dapat mengembangkan aplikasi mobile atau website menggunakan API yang tersedia.
